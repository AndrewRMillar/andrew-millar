About me

Ik ben een bioloog die en-route de weg is vergeten naar het lab. Na jaren van hobbyen op mezelf heb ik besloten van mijn hobby mijn werk te maken. Via verschillende online bronnen en cursussen zowel betaald als gratis heb ik mij in meer of mindere mate bekwaamd in html, css, python, javascript, meteor, ReactJS, VueJS, Bootstrap, handlebars.js. Ook ben ik bekend met scrum en agile.

Ik ben analytisch, gestructureerd en systematisch. Ik ben in staat mij nieuwe kennis snel eigen te maken. Daarnaast kan ik goed samenwerken met collega's maar mij evengoed uren op mezelf vastbijten in een probleem.

Style from HTML/CSS

Projectje van een aantal jaren geleden toen ik werkte bij een bedrijf dat websites scrapte. Soms moesten widgets in klanten pagina's worden gezet en was er een css conflict met de klant css. Uitzoeken waar dat conflict zat was vaak een heel gezoek. Ik probeerde met de code op deze pagina de relevante stijl uit de klant css te halen voor classen en id's die in onze widget zaten. Technieken: html/css/js/regex

Pre-Sale

Dit was een website geschreven voor een sollicitatie. Ik kreeg de opdracht om in twee dagen een resposive website te creeren van een pdf die me toegestuurd werd. Gebruikte technieken; html/css, bootstrap.

CSS grid test

Een pagina gecreerd voor mezelf om de css grid technieken uit te proberen. Bij voorbeeld het gebruik van grid-template-row en grid-auto-columns, grid-row en grid-columns. Ik was gefascineerd dat verschillende elementen zonder problemen dezelfde grid positie konden hebben.
